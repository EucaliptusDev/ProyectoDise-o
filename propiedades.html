Ejemplos de Propiedades de Fondo y Objetos en CSS

Fondo con background-image

Este fondo usa background-position, background-size y background-attachment.

Fondo Responsivo

Este fondo usa background-size: contain y se ajusta a la pantalla.

Imagen con object-fit

La propiedad object-fit: cover; ajusta la imagen para que cubra completamente el área del contenedor sin distorsionarse.

Imagen con object-fit: contain

La propiedad object-fit: contain; ajusta la imagen para que se vea completamente dentro del contenedor sin recortarse.

Imagen con object-fit: fill

La propiedad object-fit: fill; distorsiona la imagen para que se ajuste completamente al contenedor.
